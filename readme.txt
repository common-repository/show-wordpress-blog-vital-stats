== Description ==
Simple WordPress Stats Plugin to show Google Page Rank, Alexa rank, Twitter follower count, Google plus one count, Technorati rank, Number of Posts, Number of Comments, Trackbacks and pingbacks, Feedburner count etc. anywhere on your wordpress blog to show off to your visitors. 

You can show the vital blog stats and chicklets in a sidebard widget just by using a shortcode.

It is a good idea to show off some of the vital statistics of your site to your visitors (potential advertisers). If you don't want the hassle of updating your 'Advertise Here' page of your blog every few days with the important statistics then this plugin will come in handy

For information and updates, please visit the [WP Vital Blog Stats Plugin](http://www.tipsandtricks-hq.com/?p=1322) Page.

== Usage ==
There are three ways you can use this plugin:

1. Add the [wp_blog_vitals] shortocde to a WordPress post or page to show the blog stats.
2. Use the &lt;?php echo print_wp_blog_vitals(); ?&gt; function from a template (.php) file
3. Use the 'WP Blog Vitals' widget from the widgets menu

== Installation ==

1. Unzip and Upload the folder 'wordpress-blog-vitals' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings and configure the options. Set which stats you want to show off and the stat caching time etc.
4. Use the widget, stats shortcode or call the php function to show the stats to your visitors.
